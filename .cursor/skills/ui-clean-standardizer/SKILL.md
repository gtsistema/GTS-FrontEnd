# UI Clean Standardizer

Use this skill for layout cleanup, visual consistency, spacing, responsiveness, and readability.

## UI rules

1. Reuse design-kit utilities and global tokens first.
2. Keep spacing and hierarchy consistent across similar screens.
3. Prefer simple, low-specificity CSS selectors.
4. Ensure mobile behavior is intentional and stable.
5. Preserve accessibility basics (contrast, focus, labels).

## Styling checklist

- Avoid duplicated style blocks by extracting shared classes.
- Keep active/hover/focus states consistent.
- Ensure forms/tables/modals follow same visual grammar.
- Validate no overflow/crop regressions in common breakpoints.
- Keep CSS clear and easy to maintain.

## Output format

- Visual inconsistencies fixed
- Reused tokens/components
- Responsiveness checks performed
