# Validacao de contrato API

Valide a integracao frontend-backend das alteracoes atuais.

Checklist:
- Confirmar endpoint, metodo HTTP e rota utilizados.
- Conferir payload enviado (nomes, tipos e obrigatoriedade dos campos).
- Conferir mapper de response para modelo de tela.
- Verificar tratamento de erro e mensagens para usuario.
- Listar divergencias entre frontend e contrato real do backend.

Formato de resposta:
1) Conformidades
2) Divergencias encontradas
3) Correcao recomendada (objetiva)
